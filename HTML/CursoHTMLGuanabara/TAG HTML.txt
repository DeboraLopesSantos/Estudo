<!----> - tag comentario dentro html
<html lang=""> - tag para informar o idioma que sera usado
<meta charset=""> - tag usada para formatação da pagina para manter acentuações "utf-8" usasda para lingua portuguesa!
<br> - tag quebra de linha (Enter)
&nbsp; - Espaçamento dentro da linguagem html5 (Espaço)
<hgroup> </hgroup> - tag para agrupar titulo e subtitulo de um mesmo assunto.
<div> </div> - tag para delimitação da area, utilizada dentro da tag <body>Corpo da pagina</body>.
<style> </style> - tag que define estilo css, toda formatação de estilo da pagina entra dentro dessa tag (fonte, cor, tamanho, sombra,).
<header> </header> - tag cabeçalho-interface.
<p> </p> - tag abre e fecha paragrafo.
<p style="text-align: justify;"> </P> - tag utilizada para justificar o paragrafo.
<p style="text-align: justify; text-indent:50px"> </p> - tag utilizada para espaço inicial do paragrafo(visivel na pagina).
utilizando dentro das tags <head> </head> e <style> </style> deixa a formatacao de paragrafo para toda a pagina.

=> Tag Atual <body style="background-color: red"> mesmo efeito final.


***OBS: Dá para trabalhar direto dentro da tag <style> </style>
 EX:
<style>
        h1 {
            font-family: Arial;
            font-size: 35pt;
            color: brown;
            text-shadow: 2px 2px 20px black;
        } p {
        text-align: justify;
        text-indent: 50px;
        } body {
            background-color: greenyellow;
            color: red; => para definir a cor da letra
        }
    </style>

IMAGEM
/**/ => TAG COMENTARIO CSS
 COLOCAREMOS NESSE CASO /*FORMATAÇÃO DE IMAGEM COM LEGENDA*/

- PRIMEIRA COISA QUE VAMOS FAZER É IR NO ARQUIVO HTML QUE ESTAMOS TRABALANDO E DEFINIR UMA CLASSE PARA A IMAGEM 
- NO ARQUIVO NA TAG <FIGURE> COLOCAREMOS O PARAMENTRO "CLASS" => <figure class="nome que vai salvar">
Ex:
   <figure class="foto-legenda">    e salvar e ir para o arquivo estilo para formatar estilo para toda imagem que utilizar a tag figure.foto-legenda OBS; SEMPRE QUE UTILIZAR O PARAMETRO CLASS VOÇÊ VAI COLOCAR O NOME DA TAG(FIGURE NESSE CASO).NOME DA CLASSE(FOTO-LEGENDA NESSE CASO).

JÁ NO ARQUIVO DE ESTILO
 ABRIR A TAG Figure.foto-legenda(NESSE CASO) E ABRIR {}, DENTRO DAS {} VAI A FORMATAÇÃO DA IMAGEM QUE VAI USAR.

  figure.Foto-legenda {
    position: relative; /*formata posicao da legenda*/
    border: 8px solid white;  /* px tamanho da borda, solida, e cor da borda*/
    box-shadow: 1px 1px 4px black; /*sombra imagem*/
    }
     figure.Foto-legenda img {
         width: 100%; /* largura da imagem*/
         height: 100%; /* altura da imagem */
     } /* colocando img determina que toda imagem img figure a borda acompanhara o tamanho da imagem*/
     figure.Foto-legenda figcaption 
     {
         opacity: 0; /* a legenda vai ficar transparente vampos criar um efeito para sobresair quando passar o mouse por cima*/
         position: absolute; /*posiçaõ absoluta da legenda*/
         top: 20px; 
         background-color: rgba(0, 0, 0, .4); /*fundo da legenda preto com leve transparencia*/
         color: white; /*cor da legenda*/
         width: 100%;
         height: 98%; /*colocando WIDTH E HEIGTH FAZ COM QUE O FUNDO DA LEGENDA TOME TODA A IMAGEM DANDO EFEITO DE FLUTUANDO*/
         padding: 10px; /* POREM USANDO ELE ULTRAPASSA O LIMITE DA BORDA USAREMOS O BOX SIZING*/
         box-sizing: border-box;
         transition: opacity 1s;
     }
     figure.Foto-legenda:hover figcaption 
     {
         opacity: 1;
     }/*o hover vai dispara uma formatção especifica, 
     quando passar o mouse por cima da imagem/objeto (que no caso é o figcaption) que estiver com efeito transparencia*/
